Clone Google Drive dengan sqlite sebagai database, MINIO sebagai garage, dengan JavaScript sebagai bahasa yang digunakan di backend dengan library UI Tailwind CDN.
Di sini ada fitur Buat Folder, Upload File, Edit File/Folder, Delete, Pindah ke (memindahkan folder/file ke folder lainnya), file preview, search, dan bisa upload bersamaan menggunakan local (ip4 address yang sama).
